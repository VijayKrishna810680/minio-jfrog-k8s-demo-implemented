# AI Chart Board (Dashboard)

This will be a FastAPI-based dashboard with real-time charts powered by AI analytics. It will visualize MinIO bucket data and file uploads, and can be extended for more analytics.

Features:
- Real-time file upload and listing
- AI-powered analytics (file trends, predictions)
- Interactive chart board (using Plotly or Chart.js via frontend)

Instructions:
- Backend: FastAPI (Python)
- Frontend: Simple HTML/JS (can be extended)
- AI: Use scikit-learn for demo analytics, can be replaced with advanced models
